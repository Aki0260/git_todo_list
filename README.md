# Java Todo List（Console App）

Javaで作成した、コンソール上で動作するTodoリストアプリケーションです。
タスクの状態（未完了・完了）や完了時刻を管理し、
表示順を制御することで、実用性を意識した設計にしています。

## 製作背景
Javaの基礎文法およびオブジェクト指向の理解を深めるために作成しました。
既存のサンプルコードを参考にしつつ、以下の機能は自分で設計・実装しています。
- 未完了タスクを上、完了タスクを下に表示する並び替え機能

## 機能
- 新規タスクの追加
- タスク一覧の表示
- タスクの完了
- タスクの削除
- 未完了タスクを上、完了タスクを下に表示（完了時刻順）

## 使用技術
- Java
- IntelliJ IDEA
- Git / GitHub

## 実行方法
1. このリポジトリをクローン
2. `TodoApp.java` を実行
3. コンソール上のメニューに従って操作
