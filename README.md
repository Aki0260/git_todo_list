# Java Todo List（Console App）

Javaで作成した、コンソール上で動作するTodoリストアプリです。


## 製作背景
既存のサンプルコードを参考に作製しました。
その後、以下の点を自分で設計・実装しています。
-未完了タスクを上、完了タスクを下に表示する並び替え機能
-完了時刻を保持し、完了タスクは完了順に表示
## 機能
- 新規タスクの追加
- タスク一覧の表示
- タスクの完了
- タスクの削除
- 未完了タスクを上、完了タスクを下に表示（完了時刻順）

## 使用技術
- Java
- IntelliJ IDEA
- Git / GitHub

## 実行方法
1. このリポジトリをクローン
2. `TodoApp.java` を実行
3. コンソール上のメニューに従って操作

## 補足
学習目的で作成したアプリです。
